# Плагин: Таймер для VSCode
Выполнил: Батуренко Илья Дмитриевич M3114
### Описание и принципы работы:
В основе работы таймера заложен метод Помодоро. \
Классический форма: Примерно 25 минут работы над задачей и 5 минут отдыха, после 4 циклов отдыха следует один большой с промежутком от 15 до 30 минут.
### Функции таймера:
Таймер распологается в side bar`е VSCode и имеет приятный интерфейс.
Основные функции:
* Сам таймер, который динамически обновляется каждую секунду
* Уведомления для начала и конца перерыва
* Возможность просмотра статистики работы и отдыха как за сегодня, так и за все время
* Возможность изменения времени отдыха и работы, а так же возможность определить через сколько циклов последует длинный отдых
* Кнопки сброса статистики за сегодня/все время
* Все данные хранятся локально
### Инструкция для установки:
1. Склонировать репозиторий
```
git clone https://github.com/krenixx/plugin-timer-for-vscode.git
```
2. Перейдите в папку проекта:
```
cd plugin-timer-for-vscode
```
3. Установите зависимости:
```
npm install
```
4. Соберите плагин
```
npm run package
```
5. В VS Code откройте командную палитру (Ctrl+Shift+P или Cmd+Shift+P) и выберите:
Extensions: Install from VSIX, укажите путь к созданному файлу
